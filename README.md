# Kuzo Portfolio

Премиальное одностраничное портфолио frontend-разработчика Kuzo (Москва).

## Стек

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS v4
- Framer Motion

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).
